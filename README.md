# Customer Shopping Behavior Analytics

## Overview
This project analyzes customer shopping behavior using transactional data to identify purchasing patterns, trends, and business insights.  
The analysis is performed using Python for data processing, SQL for analytical querying, and Power BI for visualization.

## Dataset
The dataset contains customer-level shopping transactions, including purchase details, product categories, dates, and spending information.

## Tools and Technologies
- Python (Pandas, NumPy, Matplotlib)
- SQL
- Power BI
- Excel

## Analysis Scope
- Data cleaning and preparation
- Exploratory data analysis (EDA)
- Customer behavior analysis
- SQL-based aggregation and trend analysis
- Dashboard creation for business reporting

## Key Insights
- Certain customer segments contribute disproportionately to total revenue
- Seasonal and temporal trends affect purchasing behavior
- Product categories perform differently across time periods

## Repository Contents
- Python notebook for analysis
- SQL scripts for querying insights
- Power BI dashboard file
- Supporting documentation and presentation
